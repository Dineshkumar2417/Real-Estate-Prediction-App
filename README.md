# 🏠 Real Estate Price Predictor

This is an end-to-end Machine Learning web application that predicts property prices based on area, number of rooms, and number of occupants.

## 🚀 Project Overview
As a student exploring Data Science, I built this project to understand how to move from a raw dataset to a functional, interactive web application. It uses a Linear Regression model to find patterns in real estate data.

## 🛠️ Tech Stack
* **Language:** Python
* **Machine Learning:** Scikit-Learn (Linear Regression)
* **Data Processing:** Pandas, NumPy
* **Frontend UI:** Streamlit
* **Deployment Tools:** Git, GitHub

## 📊 Features
* **Interactive UI:** Users can input specific property details via a sidebar.
* **Real-time Prediction:** The model processes inputs instantly to give an estimated market value.
* **Model Persistence:** Uses Joblib/Pickle for efficient model loading.
