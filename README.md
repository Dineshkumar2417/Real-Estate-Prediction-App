# 🏠 Real Estate Price Predictor

A comprehensive data science project that predicts property prices based on various features such as location, area, and amenities. This project covers the entire pipeline from data cleaning to model deployment.

## 🚀 Live Demo
Access the live predictor here:
👉 [**Real Estate Predictor Live**](https://real-estate-prediction-app-ewsjaj2kvwzs4b3tuxnwgu.streamlit.app/)

## ✨ Features
- **Accurate Estimations:** Implemented optimized regression algorithms to provide reliable price evaluations.
- **Feature Engineering:** Extensive data cleaning, outlier removal, and handling of categorical location data.
- **Intuitive UI:** Clean and simple Streamlit interface for quick price estimations.
- **Scalability:** Designed to be easily adaptable to different regional real estate datasets.

## 🛠️ Tech Stack
- **Language:** Python
- **Algorithms:** Linear Regression / Random Forest Regressor
- **Data Analysis:** Pandas, NumPy
- **Frontend:** Streamlit
- **Visualization:** Seaborn, Matplotlib

## 📂 Project Structure
- `main.py`: The deployment-ready Streamlit interface.
- `house_model.pkl`: Saved regression model after hyperparameter tuning.
- `EDA_Notebook.ipynb`: Detailed Jupyter notebook covering Exploratory Data Analysis and Model Building.
- `data/`: Dataset used for training and testing.

## ⚙️ Installation
1. Clone this repo.
2. Install the required libraries: `pip install -r requirements.txt`
3. Launch the dashboard: `streamlit run main.py`
